Result: TimeoutException - Element not found ❌

AFTER: 4 fallback locators

private By searchButton2 = By.xpath("//button[contains(text(), 'Search')]");
private By searchButton3 = By.cssSelector("button[type='submit']");
private By searchButton4 = By.xpath("//button");  // Most permissive

+ Auto-fallback:
  Try searchButton1 → if fails
  Try searchButton2 → if fails
  Try searchButton3 → if fails
  Try searchButton4 → if fails
  Print debug info ✅
