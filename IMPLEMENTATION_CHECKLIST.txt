FALLBACK STRATEGY SUMMARY:
================================================================================

When a locator fails, the system:
  1. Tries locator #1 (most specific)
  2. If fails → Tries locator #2 (less specific)
  3. If fails → Tries locator #3 (even less specific)
  4. If fails → Tries locator #4 (most permissive)
  5. If all fail → Prints all available elements
  6. Throws descriptive error

This ensures robustness against minor HTML changes!
