LOCATOR STRATEGIES USED:
================================================================================

For EVERY element, we now try:

1. Original specific locator (id, exact class, etc.)
2. Generic but safer locators (contains text, partial class match)
3. Most permissive fallback (any matching element)
4. If all fail → PRINT debug info and throw error with available elements

This ensures maximum compatibility even if HTML structure changes slightly.
